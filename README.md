# React Essentials Assignment - Task Manager

## 📌 Overview
This is a Task Manager application built using React. It allows users to manage daily tasks efficiently.

## 🚀 Features
- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as Completed / Incomplete
- Responsive design

## 🛠️ Tech Stack
- React.js
- JavaScript
- CSS
- Context API
- useReducer

## 📁 Folder Structure
src/
 ├── components/
 ├── context/
 ├── reducer/
 ├── styles/

## ▶️ Run Project
npm install
npm start
